# XtsTinyBasicPlus

a fork of TinyBasic for Arduino by Xtase (fgalliat) @Nov 2016

(It's an early stage work, so it's dirty & not well documented)

* added support for :
   - ssd1306 LCD I2C Screen
   - SPI uSD Card reader
   - ESP8266 Wifi module
   - Teensy ++2 dev board

* previous authors :
